    # 堆

    重要思想：

    • 使用堆的好处是，可以处理庞大的数据，即数据不用一下子存入到内存中，并且比较的操作可以交给堆内部完成。

    大根堆，小根堆。

    构造堆：

    以构造一个大根堆为例子。

    构造堆有两种方法：1. 从头到尾一个个维持大根堆，维持大根堆的方式（在尾部先加上新加入的元素，接着与其父节点比较，若小于，则跳过；若大于，则交换，一直与父节点比较交换）；2. 从一个已经有的无序数组开始，从尾部往头部遍历，一个个维持大根堆，维持大根堆的方式（从维持的最前面的元素，与子结点比较；若大于，则不变；若小于，则交换，再与下面的子节点比较交换）

public static void heapify(int[] arr, int index, int heapSize) {
    int left = index*2 + 1;
    while(left < heapSize) { // 下面还有孩子
       int largest = left+1 < heapSize && arr[left+1]>arr[left]
           ?left+1:left;
        largest = arr[largest]>arr[index]?largest:index;
        if(largest==index) {
            break;
        }
        swap(arr,index,largest);
        index=largest;
        left=index*2+1; // left更新，还要继续比较，注意，这种写法
    }
}

    堆排序：

    1.首先将待排序的数组构造成一个大根堆，此时，整个数组的最大值就是堆结构的顶端

    2.将顶端的数与末尾的数交换，此时，末尾的数为最大值，剩余待排序数组个数为n-1

    3.将剩余的n-1个数再构造成大根堆，再将顶端数与n-1位置的数交换，如此反复执行，便能得到有序数组

    // 借助构造大根堆的操作，实现堆排序
public static void heapSort(int[] arr) {
    // 首先考虑特殊情况
    if(arr==null&&arr.length<2) {
        return;
    }
    // 构造一次大根堆
    for(int i=arr.length-1; i>=0; i--) {
        heapify(arr, i, arr.length);
    }
    // 顶端的数与末尾的数交换
    /*
    实现逻辑：
    	1. 
    	2. 
    */
    int index = arr.length;
    swap(arr, 0, --index);
    while (index) {
        heapify(arr, 0, index);
        swap(arr, 0, --index); // 考虑是否出现swap(arr, 0, 0),异或能不能行动,所以这里是错的，swap不能写成异或
    }
}
// 构造一次大根堆
// 参数：数组，从哪里开始，到哪里结束
public static void heapify(int[] arr, int index, int heapSize) {
    /*
    实现逻辑：
    	1. 当前结点和其子节点最大的比较：
    	2. 若小于，则交换
    	3. 若大于，则跳过
    	4. 当前结点往前进一个
    */
    int left = index*2 + 1; // 考虑存不存在:不存在直接跳过
    while(left<heapSize) {
        // 先判断存在，再是大小比较
        int largest = left+1<heapSize && arr[left+1]>arr[left]?(left+1):left;
        // 左右子结点与父亲结点比较，父大则跳过，父小则交换
        if(arr[largest]>arr[index]) {
            swapw(arr, index, largest);
            index = largest; // 交换后，原先index位置上元素还要和下面比较
            left = index*2 + 1; // 判断是否到底
        } else {
            break;
        }
    }
}

    # 二叉树

    # 1. 二叉树遍历

    递归：系统帮你压栈，非递归：需要自己实现压栈。

    主要理解这个栈的使用内部细节，以及它的递归序，还有一些应用利用它。

    这里注意递归和非递归的实现原理图解，主要是熟悉栈的使用。

    最终是记住这几种写法：可以记住每种的入栈方式，这样一个机制。

    注意这几种的一个互相关联的关系：

    • 前序（头左右）

      ​ 先把头结点入栈

      1. 在弹出一个结点的时候打印
      2. 如果有右孩子，先压右孩子，没有就不压了
      3. 如果有左孩子再压左孩子

      循环往复上述

    • 中序（左头右）

      • 当前节点叫做current，current为头的树，整条左边界进栈，直到遇到空
      • 上一步无法继续后，从栈中弹出节点打印，进入这个节点右孩子，记作current，继续执行上一步
      • 直到栈为空为止

    • 后序（左右头） 这个可以把先序反过来(头左右)调换为(头右左)，再存入栈中，不打印，借助两个栈（借助这种思想）

    • 层序

      • 借助队列实现
      • 先放入头结点，出就打印
      • 先放左再放右，周而复始
    // 前序遍历递归实现(中左右)
public static void preOderRecu(Node head) {
    // 若当前节点为空，返回
    if(head==null) {
        return;
    }
    // 节点不为空，刷出此节点的值
    sout(head.value);
    preOrderRecu(head.left);
    preOrderRecu(head.right);
}

/*这个比前面先序遍历难以理解
通过摸索，我的大致理解：
	- 中序按照左头右的顺序输出
	- 栈可以按照  头-左  的顺序入栈
	- 出栈   左   先出√(此时必没有它的左，也就是它是此时的头   即完成了 左头的顺序)      
	- 判断右边有没有，右就入栈，按照如上逻辑出栈输出 
上述如何实现呢？

/*实际步骤：
	- 当前节点叫做current，current为头的树，整条左边界进栈，直到遇到空
	- 上一步无法继续后，从栈中弹出节点打印，进入这个节点右孩子，记作current，继续执行上一步
	- 直到栈为空为止

// 自己理解实现
public static void inOrderUnrecu(Node cur) {
    if(cur==null) {
        return;
    }
    Stack<Node> stack = new Stack<Node>();
    while(!stack.isEmpty()||cur!=null) {
        if(cur!=null) {
            stack.push(cur);
            cur=cur.left;
        } else {
            cur = stack.pop();
            sout(cur.value);
            cur = cur.right; // 关键点：不是null执行上一步操作即把左边界全部入栈；若是null则继续执行这一步：把结点出栈再判断
        }
    }
}

// inOrderUnrecu(head)实现

            cur = cur.right; // 若有就等于right，没有就是null
        }
    }

/*
注意：
	- 思想是最重要的，清晰，代码就简单
	- 这里的思想是：先把头节点放入，出去就打印且记作cur当前结点，判断且添加左结点右结点，如此往复
*/
public static void w(Node head) {
    if(head==null) {
        return;
    }
    Queue<Node> queue = new LinkedList<>();
    queue.add(head);
    while(!queue.isEmpty()) {
        Node  cur = queue.poll();
        sout(cur.value);
        if(cur.left!=null) {
            queue.add(cur.left);
        }
        if(cur.right!=null) {
            queue.add(cur.right);
        }       
    }
}

    # 错误自查

    1. 蠢逼行为：一个while循环中，却没有停止循环的条件（例如指针不改变，不跳出来break）

      // 错误1：死循环
while(left<heapSize) {
    int largest = left+1<heapSize && arr[left+1]>arr[left]?(left+1):left;
    if(arr[largest]>arr[index]) {
        swap(arr, index, largest);
        index = largest; // 交换后，原先index位置上元素还要和下面比较
        left = index*2 + 1; // 判断是否到底
     } 
}
