    # 异或运算

    异或运算

    • 可以理解为无进位相加

    • 0^N=N N^N=0

    • 交换，结合律满足

    • 一个小技巧应用：不使用额外空间交换两个数（前提是a和b指向的两块内存是两块内存）

    • 题目：(1)一个数组中，只有一种数出现了奇数次，other偶次；(2)两种数出现了奇数次，other偶数次。

      (1)eor 从头到尾 亦或 ，最后即得到这个数
(2)从头到尾异或，最后得到a^b不等于0，即eor的某一位上不等于0，即a和b在这一位上不一样。再在这一位上异或，可以在一侧得到a或者b，得到后与第一次异或结果再异或，得到另外一个结果，即可。

    # 归并排序

    归并排序mergeSort

    • 思想

      整体就是一个简单递归，左边排好序，右边排好序，让其整体有序。

      归并排序（MERGE-SORT）是利用归并的思想实现的排序方法，该算法采用经典的分治（divide-and-conquer）策略（分治法将问题分(divide)成一些小的问题然后递归求解，而**治(conquer)**的阶段则将分的阶段得到的各答案"修补"在一起，即分而治之)。

public static void process(int[] arr, int L, int R) {
    if(L==R) {
        return;
    }
    int mid=L+((R-L)>>1);
    process(arr,L,mid); // 左侧先排好序
    process(arr,mid+1,R); // 右侧再排好序
    merge(arr,L,mid,R); // 整合到一起就排好序了
}
public static void merge(int[] arr,int L, int M, int R) {
    int[] help = new int[R-L+1];
    int i=0;
    int p1=L;
    int p2=M+1;
    while(p1<=M&&p2<=R) {
        help[i++]=arr[p1]<=arr[p2]?arr[p1++]:arr[p2++];
    }
    while(p1<=M) {
        help[i++]=arr[p1++];
    }
    while(p2<=R) {
        help[i++]=arr[p2++];
    }
    // 注意这里的将得到排序好的数组弄到原数组上。
    for(i=0; i<help.length; i++) {
        arr[L + i] = help[i];
    }
}

        class Solution {
    public int reversePairs(int[] nums) {
        if (nums==null || nums.length==0) {
            return 0;
        }
        return process(nums, 0, nums.length-1);
    }
    private static int process(int[] arr, int left, int right) {
        if(left==right) {
            return 0;
        }
        int mid = left+((right-left)>>1);
        return process(arr, left, mid)+process(arr, mid+1, right)+merge(arr, left, mid, right);
    }
    private static int merge(int[] arr, int L, int M, int R) {
        int[] help = new int[R-L+1];
        int p1 = L;
        int p2 = M+1;
        int result = 0;
        int i = 0;
        while(p1<=M&&p2<=R) {
            result+=arr[p1]>arr[p2]?(R-p2+1):0;
            help[i++]=arr[p1]>arr[p2]?arr[p1++]:arr[p2++];
        }
        while(p1<=M) {
            help[i++]=arr[p1++];
        }
        while(p2<=R) {
            help[i++]=arr[p2++];
        }
        // 不能定义i???
        for(int j = 0; j<help.length; j++) {
            arr[L+j] = help[j];
        }
        return result;
    }
}

    # 快速排序

    快速排序：

    由荷兰国旗，相关过来

    • 不改进的快速排序：

      • 把数组范围内的最后一个数作为划分值，然后把数组通过荷兰国旗问题划分成三个部分：左侧<划分值；中间=划分值；右侧>划分值；对于左侧范围和右侧范围，递归执行。
      • 划分值越靠近两侧，复杂度越高；划分值越靠近中间，复杂度越低
      • 可以很简单地举出最差的例子，所以不改进的时间复杂度为O(N^2)。

    • 快速排序1：O(N^2)。最差情况。

    • 快速排序2：O(N^2)。最差情况。

    • 快速排序3：O(NlogN)，通过随机选择一个数做划分，经过数学证明为O(NlogN)。证明和概率有关。空间复杂度最差为O(N)此时选择的是终点的位置，最好为O(logN)此时选到的是接近中点的位置。

        public static void heapify(int[] arr, int index, int heapSize) {
    int left = index*2 + 1;
    while(left < heapSize) { // 下面还有孩子
       int largest = left+1 < heapSize && arr[left+1]>arr[left]
           ?left+1:left;
        largest = arr[largest]>arr[index]?largest:index;
        if(largest==index) {
            break;
        }
        swap(arr,index,largest);
        index=largest;
        left=index*2+1; // left更新，还要继续比较，注意，这种写法
    }
}

    # 堆排序

    堆排序

    • 思路：

      • 主要是借助heapInsert和heapify操作。在堆结构的基础上。
      • 先有一个无序数组，先让数组整体变成一个大根堆（类似于用户一个一个给你数据，你把heapSize渐渐地扩大到数组整体），接着再把顶部的数(最大值)和末尾数据交换，堆大小减小1，再执行heapify操作让剩下的数变成真正的大根堆(时间复杂度为logN)，递归实现。
      • 注意这里的堆排序：因为是遍历式地执行hepify操作，所有在执行这次之前就已经实现了初步的堆，在此基础上执行heapify就ok。需要理解。

    • 代码实现：

      public static void heapSort(int[] arr) {
    if(arr=null && arr.length<2) {
        return;
    }
    
    // 1. 这里只是展示这个用上了，以下2是更加好的方法
    // 先执行一轮heapInsert,将数组变为大根堆 
    for(i=0; i<length; i++) { // O(N)
        heapInsert(arr, i); // O(logN)
    }
    
    // 2. 这个完成之后也是大根堆，常用这个方法
    for(int i=arr.length-1; i>=0; i--) {
        heapify(arr, i, arr.length);
    }
    
    // 首尾交换
    int heapSize=arr.length;
    swap(arr, 0, --heapSize);
    while(heapSize) { // O(N)
        heapofy(arr, 0, heapSize); // 在0位置上执行此操作，截止heapSize位置上 // O(logN)
        swap(arr, 0, --heapSize); // O(1)
    }
}

    // 例子

public class GraphGenerator {

	// matrix 所有的边
	// N*3 的矩阵
	// [weight, from节点上面的值，to节点上面的值]
	public static Graph createGraph(Integer[][] matrix) {
		Graph graph = new Graph();
		for (int i = 0; i < matrix.length; i++) { 
			// matrix[0][0], matrix[0][1]  matrix[0][2]
			Integer weight = matrix[i][0];
			Integer from = matrix[i][1];
			Integer to = matrix[i][2];
			if (!graph.nodes.containsKey(from)) {
				graph.nodes.put(from, new Node(from));
			}
			if (!graph.nodes.containsKey(to)) {
				graph.nodes.put(to, new Node(to));
			}
			Node fromNode = graph.nodes.get(from);
			Node toNode = graph.nodes.get(to);
			Edge newEdge = new Edge(weight, fromNode, toNode);
			fromNode.nexts.add(toNode);
			fromNode.out++;
			toNode.in++;
			fromNode.edges.add(newEdge);
			graph.edges.add(newEdge);
		}
		return graph;
	}
}
