    陆盛涛 2021/12/25 book

    # 1. 豆瓣书友“林希曦”的书评

    每每有人问我爱好是什么的时候我总是说喜欢看书。听上去求知若渴的样子。闻者当即对我赞许有佳，然后第二个问题是爱看什么书？这时候我开始结结巴巴：我，我爱看爱情故事。对方当即恍然，哦，爱看言情小说哪。

    一说言情小说就令人想到了琼瑶奶奶岑凯伦阿姨想到了情深深雨濛濛我爱你你爱我纠缠不清的眼泪要死要活。立刻形象大毁。从仿冒的知识女性沦落到了家庭妇女口味。 或许该搬点杜拉斯村上春树米兰昆德拉出来冒充小资。这不算难，我可以背诵情人的开头和结尾。或者讨论一下川端康成。卡夫卡也可以。父亲书橱里有他们的书，我可以事先预习一下。 或许这样才可理直气壮地表示喜欢看书？

    然我确实是喜欢爱情故事的。我还是打算实事求是地暴露我是个大俗人没品味的事实——我通常睡前读的都是爱情小说。我通常对风花雪月有无比的兴趣。即使看着霍去病传，心里都会暗暗可惜那么英伟神武年少得志功高权重冷峻刚毅奇男子，怎么一生都没有一段爱情的传说？

    看爱情故事的历史由来已久。9岁那年自以为认的字够看长篇小说了，父亲看什么我也跟着看什么，那时候他看的是梁羽生和金庸。我一跤跌进武侠世界里，潜意识是拿它当爱情故事看的，但凡武打场面都一目十行匆匆掠过，杨过招惹了多少女生对她们分别以怎样不同的态度却能如数家珍。 少女时代以为金庸写情至深，杨过的十六年等待可算爱情典范。

    虽然我当年也曾追随着琼瑶奶奶的脚步手捧《浪花》哭的眼肿鼻塞——后来就非常诧异，当年哪里来的那么多眼泪！事实上琼瑶笔下的爱情故事就象一幕搞笑剧，假的可笑，笑得起鸡皮疙瘩无法忍受。硬要我再看的话，大概需要胃药。

    自从我看见亦舒之后就自以为是自白垩纪进化到了侏罗纪。亦舒的文风冷硬，刻薄辛辣，女人的强悍发挥到了尽头，爱情在她笔下永远不是生活的必需品。亦舒的智慧通透在书中随处可见。 还有李碧华黄碧云叶倾城们，都是同类。都是聪慧的女子写出的清醒的爱情故事，钢筋水泥丛林里的爱情故事，合了现实性，灰色的晦涩的，令人心灰的故事。 常常是分离。淡淡的口吻诉说一次邂逅一场相知，然在现实前爱情退让，在记忆里成就永远。人们爱的是一个人，与之结婚生子的是令一个人。于是《蔷薇泡沫》中马宝琳和王子分头结婚，《两个女人》中施名扬与思龙各自想念。《人淡如菊》“我”在真正的恋爱之后嫁了另一个男子。 不是不爱，只是更爱自身。

    看多了，有一点倦。这样的爱情故事适合少女学习使用，有助于她们健康成长。我已经够清醒够冷静，用J的话来说我们都快得爱情疲乏症了。R说，也只能在别人的故事里放肆地叹一口气了。 在别人的故事里，在爱情故事里，让我重拾一下感动吧。

    我看见王小波的《绿毛水怪》的时候，我感动了。我看见了最美丽的一个爱情故事。 王小波以一种第三者回忆叙述的方式来讲这个故事，故事的叙述者是老陈，听者是老王，也就是王小波。老陈向王小波讲述了和一个叫妖妖的女孩子自童年开始的爱情。

    老陈和妖妖是青梅竹马的同学，在一次被老师的处罚中结下了友谊，两个在旁人眼中奇怪“复杂”的孩子，一起攒钱买书一起看书，彼此认为是永远的朋友。老陈说，妖妖，你最好变成男孩子就好了。妖妖说，我家没有男孩子，我要爸爸收你做我弟弟。那样的友谊真叫人羡慕向往，真正无关性别的纯粹的友情。（看到这里我就遗憾我童年的时候怎么就没人拿我当男孩子对待呢）

    后来老陈和妖妖进入了青春期，友谊渐渐变成了朦胧的爱恋。老陈天天送妖妖回家，一路上谈论诗歌，文学和理想。真喜欢这一段，他们互相喜欢，可是没有一句暧昧的言语，没有一点欲语还休的扭捏，还是纯粹坦然，与其说互相吸引，不如说互为知己。

    很多爱情故事都喜欢让男的一个劲地追女的一个劲地跑，非得男子出尽百宝用尽花招才能功德圆满，爱情在这个时候开始向技术性发展。又或者是双方都慧眼明察一进一退地试探，言语中尽藏机锋，十分伶俐精彩，可是在老陈和妖妖的爱情面前，一切似都褪色。最好最好的那种爱情，应该就是这样，似水晶般剔透的两心相知。无人计较谁先开始爱，谁爱更多一点，无关技巧没有杂念，真正的赤子之心。

    文化大革命开始后令老陈和妖妖分散了几年，老陈非常非常想念妖妖，这时候他明白了妖妖对他来说的真正意义。但是等他终于找到她的时候，她已经在游泳的时候溺水身亡了。 故事到这里没有结束。老陈在海里游泳的时候遇见了一群绿毛水怪——碧绿的身子，像海藻一样的头发，像海豹一样的四肢。。。。。形状丑恶狰狞，令人害怕而且恶心。他们原本都是人，服用了药物后才变成了水怪。虽然变不回去了，可是他们在海底建立了王国，在海洋称王称霸科技发达。他们邀请老陈加入，老陈一口拒绝了，他从来不吃鱼闻到水怪的腥味都恶心。水怪也不勉强他也不要他保密，反正说出去也没人相信。当老陈要和水怪分手的时候，他突然发现一个女水怪就是妖妖。 妖妖说我已经变不回去了水怪离开水24小时就会死的。老陈说你怎么知道我不愿意和你一起当水怪呢？ 老陈当即表示要立刻服下变水怪的药。可是药在其他水怪身上，妖妖要去拿药，约好老陈第二天在这里相会。谁想老陈当晚回去就发起了高烧昏迷被关进了医院怎样都脱不了身，等到他终于从医院逃出来赶到相约的小岛上，只看见妖妖在石头上刻下的留言：陈晖，祝你在岸上过的好，永别了。但是你不该骗我的。杨素瑶。

    老陈的故事说完了。老陈悲痛难抑眼中含泪，王小波以不相信的态度结束了小说。“你们见过这样的人吗，编了一个弥天大谎，却硬要人相信？甚至动手打人！可是我挨了打我打不过他被他揍了一顿。。。。。。世上还有天理吗？”

    《绿毛水怪》是王小波的早期作品。无疑那时候王小波是相信爱情的，或者说，在心底是相信爱情的，所以才能编出这样的一个故事。但在爱情式微的年代，还有多少人会相信爱情呢？哪怕只是听一个故事，有多少人愿意相信愿意感动呢？于是他做出与大众一般不置信的姿态，一般感觉荒谬的姿态。

    人们不愿相信的，到底是绿毛水怪这种东西，还是爱情呢？

    这个故事是我见过最自然最深情的爱情故事。我根本不用去考虑水怪是否真正存在，这完全不重要。我已经感动了，我愿意相信世界上有这样的爱情，因为爱，愿意做一只水怪。 这是真正意义上的爱情。王小波通篇没有一句煽情没有一句海枯石烂的誓言花前月下的甜言，用最朴素最平淡的语言，却说尽了深情。许仙看见白娘子的真身就弃之如敝，老陈却愿意为妖妖做一只最丑陋的水怪。 杨过若看见小龙女成了一只水怪，还会爱她么？也会愿意做水怪么？ 童话故事里美女爱上野兽，一个吻将它变为王子。若要美女也同样做一只野兽，她愿意么？

    更重要的，是爱的自然。爱不是牺牲，需要牺牲的爱情不是真爱情。老陈愿意做水怪不是为了妖妖牺牲，他做了水怪和妖妖在一起才是真正的快乐。爱情不能有一丝的勉强。若需要权衡利弊需要左思右想，所有的牺牲都是后悔的隐患。 需得两心如一。

    当然我知道这个故事是王小波编出来的。他编的自己都心虚得以不敢相信的姿态出现。可是我仍然愿意相信这样的爱情，纯洁，坚贞，不顾一切，两心如一。

    《绿毛水怪》是一篇爱情的桃花源记。没有人知道桃花源在哪里，没有人会真正去寻找桃花源。因为它不是一个可以寻找的地方。可是它那样美丽那样令人安慰。

    每个人心底深处，是否都有一个桃花源？

    # 2. 豆瓣书友“晶”的书评

    不需要很长很华丽的语言，王小波也可以把故事讲好。 如果说最俗的故事是爱情，那么最惊艳的故事也是，而且往往是爱情。“绿毛水怪”的故事很简单，只言片语，但很动人。

    就像那句“你是我军旗”点亮了我眼中的《爱你就像爱生命》一样，《绿毛水怪》里有好多字句给人心头一击。 比如，童年时，妖妖和陈辉甘愿做两只如“怪物”，去旧书屋分享好书。 比如，初中相遇，妖妖绝口不提他们儿时最爱的《雾海孤帆》，陈辉选择放弃课后一切活动送妖妖回家，听她用无法形容的语气背出《青铜骑士》的楔子：我爱你，彼得建造的大城…… 比如，最后一次在有雾的夜晚见面，陈辉为妖妖念出诗句，告诉她街头的水银灯是“大团的蒲公英浮在街道的河流口,吞吐着柔软的针一样的光”，而昏黄路灯是“我们好象在池塘的水底，从一个月亮走向另一个月亮”。 比如，妖妖留给陈辉的信，说他们忠贞不变的友谊是六百四十二个来回，和两千五百里路见证的。 再比如，陈辉对绿毛海怪脱口而出的那句“……你怎么知道我不愿意和你一起当水怪了呢”？

    正如李银河说，王小波可以像一个诗人，虽然他更多写的是杂文和小说。的确，他语言嬉皮，压抑着癫狂，但全然遮掩不住他诗人一样如空气浮动般的气质。 我喜欢妖妖自我鉴定只能做一个“蓝袜子”，而赞美陈辉可以成为诗人那段对话。我赞同王小波书里常常流露出的“女性对男性保有宿命式崇拜”的观点。在某种程度上，我很迷恋王小波这种霸道，区别于弗洛伊德所描述的那种男根崇拜的霸道，王小波的文字里终保有了一种温柔的态度，对男性和女性同时保有最大的尊重。所以读完《绿毛海怪》，你不会像大多数爱情故事一样，只记住一个妖妖、或是一个陈辉，相反你会同时记住他们两个人。王小波写故事难得之处就在于他笔下的爱情从来都不只是一个人的事情，你不会单纯唏嘘妖妖没有多等陈辉两年就游向了大海深处，不会质疑陈辉一而再的错过和妖妖见面的机会，你只可以感慨社会，时代和命运弄人——爱情本质无关是非，有关平等包容，是王小波对爱情的解读。

    还有一些很有意思的，这本小说是王小波早期作品，写于他与李银河相遇之前（后来是李银河和王小波相遇相知相恋的重要桥梁）。这就不难理解，书中的“我”为什么会时而跳出来打断老陈（陈辉）讲故事，戏谑妖妖的故事太过荒诞。这一时期的王小波在很大程度上，对自己笔下的崇高爱情所持的态度显然抱有悲观态度，不可遇也不可求。

    最后一点想说，这个故事有关海洋…自由，广博又宁静。

    PS：看完书，也想打这样的一些字给远方的人。你说过我是最好最认真的海棠，翻出这个故事让我再看，非常感谢你相信我可以解读这个故事接近你到无穷小。当然我不似你会创造故事，因为上帝不是女人，只可以有犀利哥哥眼神似箭，却鲜有剽悍姐姐目光如炬。感叹王小波早逝这么多年，我觉得他的人和他的文字真的好像脱离时光轴线，随时取出翻看，都鲜活如现世。总之，我真的喜欢他对事物的理解、包容，和他俏皮若愚的诗意语言。

    # 3. 原文摘抄一

    我跟你说，世界上就是小孩好。真的，还不如我永远不长大呢。

    小孩子的视角，总是单纯无害的。小孩子的想法都很天真浪漫。

    # 4. 原文摘抄二

    ​ 老王，你看学校就是这么对付我们：看见谁稍微有点与众不同，就要把他扼杀、摧残，直到和别人一样简单，否则就是复杂！

    ​ 这里可以看见我们教育出现的问题。好的教育是需要尊重个体差异的，而不是千篇一律的一个固定的标准，还暂且不论这个标准是否合适。

    ​ 简单，复杂，这两个小孩在小学时就特立独行，有点自己的原则，自己的思想，大人们就要赋以“复杂”的称谓，企图扼杀。

    # 5.原文摘抄三

    ​ 我永远也忘不了叶菲莫夫的遭遇，它使我日夜不安。并且我灵魂里好像从此有了一个恶魔，它不停地对我说：人生不可空过，伙计！可是人生，尤其是我的人生就要空过了，简直让人发狂。还不如让我和以前一样心安理得地过日子。

    ​ 这让我想到了做一只幸福无脑的猪还是一个痛苦明智的人。如果你不知道智慧为何物，那么你当然可以心安理得的过日子。这个世界本也不是非黑即白，我在想，做幸福无脑的猪的也不一定不是一种大智慧。但是也有亚里士多德这样的“吾爱吾师，吾更爱真理”。哲学家一辈子在探寻真理，它们思考的东西有价值吗？那么是做一个幸福无脑的猪好呢，还是做一个痛苦明智的人？这个问题我暂时没有答案。

    # 6.原文摘抄四

    ​ 我说：“妖妖，你看那水银灯的灯光像什么？大团的蒲公英浮在街道的河流上，吞吐着柔软的针一样的光。”妖妖说：“好，那么我们在人行道上走呢？这昏黄的路灯呢？”我抬头看看路灯，它把昏黄的灯光隔着蒙蒙的雾气一直投向地面。我说：“我们好像在池塘的水底。从一个月亮走向另一个月亮。”

    ​ "我们好像在池塘的水底，从一个月亮走向另一个月亮"。这是未读此文就先知道的句子，当时只觉得很浪漫。

    ​ 能写出如此有诗意的句子，作者本人是多么的有诗意啊，“一身诗意千寻瀑”。我在想，王小波身上的诗意我们可能是无法企及获得的，但是如何让我们也有那么一点的诗意呢？也就是说，他是如何这么有诗意。我们可以浅显地分析一下他。

    ​ 王小波的精神世界就是浪漫的，他的这篇《绿毛水怪》体现了他单纯美好的爱情观。另外，他的诗意和它肆意的想象力扯不开联系。

    注释:今日回头看到上述我的文字，恰好验证了我缺乏诗意.

    # 《战福》

    # 三. 个人感悟

    # (1)关于“绿毛水怪”

    ​ 首先说出我的联想吧：1.虚幻，最后分不清是爱情的虚幻还是故事的虚幻； 人们不愿相信的，到底是绿毛水怪这种东西，还是爱情呢？2.作者的爱情观是单纯的，他以时间为线，述说了最为纯洁的感情，不矫揉造作，没有技巧没有其他的杂物。3.爱情的桃花源记，他不存在，你的心底有这样一座桃花源吗。4.王小波通过极致朴素的语言，没有华丽的辞藻就说出了真正的爱情。
