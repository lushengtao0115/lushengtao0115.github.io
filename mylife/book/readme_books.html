    我的书单和看书计划

    陆盛涛 2021/12/23 book

    这里主要记录我的书单，看过的没看过的，想看过的，以及标记出想要进行书评的书籍。

    # 书单

    # 1.《绿毛水怪》王小波

    短篇小说集

    王小波的充满诗意的书

    重读+书评

    # 2.《局外人》加缪

    短篇小说

    世界与我

    重读+书评

    # 3.《撒哈拉的故事》三毛

    短篇小说集

    关于爱的故事

    # 4.《简爱》

    中篇小说

    一个坚强独立的女性，勇敢追求

    # 5.《月亮与六便士》毛姆

    中篇小说

    唯独你看见了月亮

    # 6.《认知觉醒》

    有关认知能力的工具书

    重读+书评

    # 7.《穷爸爸富爸爸 》

    财商方面入门书

    重读+书评

    # 8.《不安公主》

    短篇小说

    探讨生命本质

    # 9.《许三观卖血记》

    长篇小说

    关于生活和苦难的故事

    # 10.《夜莺与玫瑰》

    写给成人的童话集

    # 11.《无声告白》

    中篇小说

    家庭关系

    # 12.《质数的孤独》

    中篇小说

    说孤独

    重读+书评

    # 13.《格兰贝的年轻人》

    短篇小说集

    异国情调的年轻人的故事，荷尔蒙，青春，暴力，热血

    # 14.《晚熟的人》莫言

    短篇小说集

    我读的第一本莫言的书

    读完+书评

    # 15.《寻欢作乐》毛姆

    小说

    # 16.《柒》文珍

    短篇小说集

    探讨露骨的爱情

    读完+书评

    # 17.《陈果的人生哲学课》陈果

    哲学入门书籍

    # 18.《西方哲学十五讲》张志伟

    搭配哲学十五讲视频，西方哲学入门。

    # 19.《哲学家都干了什么》

    较为全面的了解哲学历史，哲学家历史的一本书，较好的入门书籍

    # 20.《内向性格的竞争力：发挥你本来的优势》

    心理学书籍，关于内向性格的解读，较好的入门书籍。
