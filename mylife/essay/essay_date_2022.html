    每日随笔(2022年)

    陆盛涛 2022/1/2 essay

    2022年的随笔记录，坚持每日更新。

    包含生活的方方面面，可以从中总结一些东西到专项中去。

    每一项还是分项写吧，非日记形式。

    熟人看到不要告诉我，谢谢。

    陌生人看到，没关系，你不认识我（应该还没有陌生人会看到我吧）

    # 2022/1/2

    ​ 有一件事情很难：将一件困难复杂的事情简单明了地叙述出来。这里还可以思考一个问题：

    ​ 一鼓作气！有一个原则：增量式工作！

    ​ 正如老师所说的，项目整合，沟通能力也需要培养，例如对于项目中技术的实现，可以找相关专业人士实现而不需要亲历亲为。

    # 2022/1/3

    ​ 今天，做PPT，熬过去了就好了。大致事情都是如此，难易占比符合八二定律。

    ​ （16:00-22:00 6个小时)(22:40-02:00 3个小时)

    # 2022/1/5

    ​ 随心所欲不逾矩。

    ​ 最近感觉心里有点乱，慢慢来吧，好好梳理一下自己，整理自己。最近想述说的有：关于人际关系，交流，做事，自己，修养身心。我与世界。

    # 2022/1/6

    ​ 思考一个问题，不要逃避。被动式地听视频学习，似乎效率不是很高，因为你很舒服，你习惯被动接收，你缺少了主动思考的机会。另外，笔记是用来复习整理的，而不是只放在那里。

    ​ 许多东西从我的脑海里面溜走，强迫症的我想找一个全平台通用的东西来随时随地记录一些想法。或许是突发奇想，或许是看书时魂游。这应该是个不错的想法。Anyway，行动起来。

    # 2022/1/7

    ​ 打破沙锅问到底，求知欲。

    # 2022/1/8

    ​ 避免精神内耗，我很好，我感觉在这上面我还是有很大的进步。

    # 2022/1/10

    ​ 读书也是要给自己定一下小小的计划和目标的，这样才不会偏离预定的轨道。还有，读书不要老是这看一本，那看一本。

    ​ 关于读书，一方面是可以受到熏陶，从中学习；一方面可以启发自己的思考。一千个读者就有一千个哈姆雷特，打开自己的思想之门。

    ​ 不要用战术上的勤奋掩盖战略上的懒惰。今日，有件事情对我优点启发。最近在复习动态规划部分，脑子里就在想多看几题把这个点搞懂，却忽略了一题一题搞透的重要性。说到底，是因为我本能地避免了深度思考，太浮躁啦。

    # 2022/1/11

    ​ 有时候也不要太逼自己，总想着一次就做好，做完美，留着点等下次做吧，这样也有个目标，有个前进的动力——深夜有感（估计是熬太晚了，这是部分原因）。

    # 2022/1/12

    ​ 你的追求承载着你生命的重量。生活并非理想的乌托邦，我走的每一步必然也不是全考虑自己。相信我吧，这不会是我的压力，这只会让我一直前进，前进！

    # 2022/1/17

    ​ 能够解决问题的确值得高兴，可是也需要注意效率。最近在自定义博客的样式，我以为自己找对了方法，直到今天我发现了更加简单的方法(通过文档)，之前一直没有注意到，导致浪费了不少时间......

    ​ 时刻保持清醒，是过好人生的第一步，也是很难的一步。比如说，分得清楚眼前利益和长远的利益。比如说，知道别人说话没有什么坏的用意他本性如此，还是不坏好意

    ​ 有时候需要一股虎劲，不达目的不罢休！也就是说，凡事，不要老是想着退路，比如这件事情今天应该做不完了明天再做吧。

    ​ 时间真是一件重要的东西呢，还有啊，焦虑没有用。

    ​ 越在家越能感受到父母的辛苦和不易，我还有什么理由不努力？问自己。

    ​ 做事，一定得分优先级。我现在感觉，之前花了不少时间研究项目的事情，有点浪费时间了，今后把重心放在学技术找工作上面，其他的过了就行。

    # 2022/1/22

    下面是最近心里冒出来的一些想法，并没有完整的写出来，后面围绕着这些慢慢去写，整理。

    • 独立思考。
    • 寒假应该怎么做。这几天的反思，接下来怎么做。
    • 关于实习，笔试面试，学习情况等小结。
    • 摇滚，摇滚乐是什么，等等？来自于B站up主周开山，学习与思考。
    • 和别人聊天闲聊聊什么，主要是异性朋友太少了。
    • 程序员如何学习，进阶。跟着视频学，是能获得一些知识，但是发现自己只能被动接受它了，缺乏主动思考，整合。另外缺乏自己动手实践。因此要平衡好各个部分的时间，作为一个好的程序员。
    • 我在做一件事情的过程中，很容易被另外一件事情打断，等回过神来，已经过了好久。

    # 2022/1/23

    ​ 现在是23日的凌晨，脑子里面突然冒出一个大胆的想法，没日没夜的拼命肝学习！？。！

    ​ 我心里很急躁，急于求成，焦虑。但是啊，很清楚的一点是，一口气吃不成一个大胖子，我只能一步一步来，一个完善的计划，切实的执行，终会到达彼岸，做到你想做的事情。

    ​ 写这些学习博客是为了记笔记，而记笔记是为了方便日后查阅和复习，刚才看了下我之前所写的，才发现格式不统一，很乱，这我当然是能改就改啊。故后续边复习回顾，边改善格式。

    ​ 逃避，逃避开始，逃避现实，也就是在掩耳盗铃......值得的事情必定不是想象中的一帆风顺。

    学习阶段小结，更新完善。

    1. 拔苗助长无益处。于我而言，还是尽量多花费些时间和精力弄懂技术方面的东西，单纯的听视频不带脑子，不深入是不行的。保持旺盛的技术生命力！

    2. 学习时间必须不能做别的事情，必须。电视就不要开了，以前的坏习惯，强迫症，及时纠正。一直在尝试专注。

    3. 要有魄力和毅力，另外学习的时候，需要明白接下来的几个小时要干什么事情，不能不清不楚。

    4. 劳逸结合规律，激发最大的潜能。

    5. 笔记的效用要发挥，比如复习整理、查阅。万万不可受制于此。

    6. 学习资源的搭配要合理。比如书籍，若是简单翻阅就看通俗易懂的，而需要深入钻研的就看较权威深刻的。视频只是作为获取知识的一种途径，不能完全依赖，适当的时候看。

    7. 注重增量式工作，不做无用功。

    8. 不要强迫症，会的就跳过，简单的就不看视频，提升学习效率。

    9. 注意一点，人遗忘是很快的，那么之前学习的东西价值在哪里呢？这也会让我们思考，该如何学习？

    10. 一种好的学习模式：集中看视频理解，随手记下不理解的地方，完成后集中整理笔记；看视频学习前可以预习一下教案或者相关资料。

    11. 学习一门技术，更高级的一种方法是，知道为什么学习这门技术，它主要解决了什么问题？缺它不可吗？

    12. 不焦虑，掌控好自己的节奏，别人学得好进度快只会激励着自己。从0到1的过程，规律，你得接受。

    13. 学习，注意不断地迭代更新自己，局部最优 --> 全局最优 迈进。

    14. 技术：简单的看文档即可，难得主要是理解思想，还有熟练度，有些需要必须记住，有些知道，会查文档即可。还要考虑自己的遗忘规律。

    15. 充分利用好已经做过的工作。例如以前的笔记，复习整理的时候好好利用，查缺补漏。

    # 2022/1/25

    ​ 个人感觉作息时间还是调过来吧，这样可能可利用时间多一点。

    # 2022/2/12

    ​ 寒假即将结束。在家保持自律很困难呐。这个寒假学会了做几道菜，下了几次厨，几乎每天学习，就是强度降低了，边学习边看剧，orz不然真的学不下去(内心：安慰自己，不要焦虑...)，其余时间就是睡觉和刷剧。

    # 2022/2/22

    ​ 今天想要深入思考一下：如何正确地分配时间在很多事情上，想做的事情很多。还有寒假的一点小结。

    ​ 学习方面的一些切实可行的计划，需要整体的长期的，以及短期，每天的计划。

    ​ 另外，将这些专项的收录整理起来。

    # 2022/2/23

    待总结。

    ​ 不劳无获。

    ​ 别太满。

    ​ 成熟——

    ​ 有则改之，无则加勉。不必对自己感到抱歉，不好意思。

    ​ 思想的转变，或许是最为关键，且最容易忽略的。

    ​ 对自己对他人怎么做。

    ​ 坚持自己的原则，不举棋不定。

    ​ 说做就做，绝不拖延。拖延，安于现状，是慢性死亡，是一种恶习，我已经有点了。

    ​ 学会共存，与困难。

    ​ 爱具体的人。

    # 2022/2/28

    ​ 二月最后一天，小小总结一下。

    ​ 调整自己到最好的状态，迎接每一天，每一天都是崭新的一天。仔细想想，自己的态度决定自己这一天是什么心情，其实外界什么都没变。

    ​ 这几天作息出现了问题，不怎么规律，故今天开始，需要忍着白天不能休息太长时间。晚上按时睡觉，逼着自己睡够七个小时再起床。

    ​ 还有一点感叹，时间实在是过得太快了。

    ​ 一天的事情提早规划好，白天的无意义时间尽量减小到最少，尽量不一大段时间做无意义的事情。可以尝试着类似于番茄钟的做法，比如学习25mins休息5mins，学习3hours休息20mins，这样就有了片段时间去把握做其他的事情。

    ​ 学习效率还需要提升，提醒自己学习的时候不能做其他的事情。

    ​ 这几天学习目标定的有一些没有完成，但是这些都是较为重要的，原因还是整体的时间规划没有做好，有些内容需要花费大量的时间。现在改进思路是：①参照学习计划②列出学习大纲即可，一件件的去做，根据自己的状态去调整学习。

    ​ 另外，虽然说，需要学习很多方面得内容。但是在学习一项内容时，依照情况而定，如果状态很好，就多学习一点。

    # 2022/3/2

    ​ 很多事情并不是那么难，就看你愿不愿意去花时间，动脑筋，你不去做当然不行咯。

    # 2022/3/3

    ​ 及时止损思维。

    ​ 种一棵树思维。

    # 2022/3/4

    ​ 多去朋友身上学习某种特质。这样，也可以发现别人身上的优点啊，多多欣赏别人。

    ​ 学习时间的分配，什么时间段适合做什么事情，事先做好划分是最好的。

    ​ （重点）需要重点思考一个问题：我记得这些笔记意义在哪里？是不是要极致地发挥它的价值。

    ​ （重点）还需要思考一个问题：视频教程讲完这些只花了几天，而我们拉长战线花很长时间是否合适，我觉得有时候可能需要加把劲，拿出那种精神，一个阶段尽快结束。

    ​ 明白一个概念，不要只顾着走，有时候抬头看路也是好的。

    ​ 今日有感而发，学过了一段时间的东西，很长时间不碰，就忘记了。摆脱遗忘的方式：有用的东西，经常使用。没事的时候，多多复习。

    ​ 累了就歇会儿。走起来，一定能走到终点。

    ​ （重点）我发现嗷，我还是学习效率有点低了，做了不少无用功，还是要有魄力啊，不要思维定式。学习，或许要有全局观，不局限于这一个。动脑子动手。

    # 2022/3/5

    ​ 错过的不再来。

    ​ 君子慎独，不欺暗室。

    ​ 时间管理意识。

    ​ 以后准备啊，每天晚上睡前，push刷新一次博客。

    ​ （重点）每次纠结，犹豫不决，不知道如何学习，大多数情况下，都是觉得时间不够，很麻烦，不知道啥结果，一这样想就不尝试了。所以呢，凡是有办法都先试一下。尝试了，可以再回过头来看看，不行就优化吗，也不会在错误路上走的太远，而且还会对自己有所启发走好下一步，还能缓解焦虑，何乐而不为？

    ​ （重点）需要及时感知自己的状态，便于将自己的学习效率激发到最高。

    # 2022/3/6

    ​ （重点）感觉心里有个结，有些知识点，不看视频学习好像不踏实。需要清醒认识，学习知识的方式什么都可以，只要掌握到需要的程度就行，可以快的地方就快，可以慢的地方就慢。

    ​

    # 2022/3/7

    ​ （重点）焦虑的原因，对于未来有期望，但是看不清前面的路。相对可以接受的解决方法：整体计划，各个阶段应该干什么，达到什么目标，清晰明了，再就是脚踏实地，一步一步走，伴着憧憬。脚踏实地，仰望星空。

    ​ ???今天必须有个说法必须要弄清楚，学习有必要按照方式分嘛，一是看视频，而是看博客，三是看书，四是敲代码，因为状态不一样，什么时候干什么事情可能才能达到最佳状态。

    ​ 有理想型，那么自己也要成为足够优秀的人啊。哈哈，一见如故。

    # 2022/3/14

    ​ 一往无前，有信仰，有力量。请不论发生了什么，都不要丧失生活的勇气！
